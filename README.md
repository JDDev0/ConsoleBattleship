## Compile & Run
### Linux
- Required packages: `cmake`, `make`, `gcc`, `libncurses-dev`

1. `cmake -DCMAKE_BUILD_TYPE=Release -S . -B release`
2. `cmake --build release`
3. `release/Battleship`
